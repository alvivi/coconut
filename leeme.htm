1.INTRODUCCIÓN

Coconut es una versión del clásico juego 'pacman' disponible para Windows y Linux.

2.COMPILACIÓN

Coconut esta programado usando el lenguaje C y el compilador GNU GCC (Aunque puede ser compilador por cualquier otro compilador de C). El juego usa las librerías de videojuegos SDL, además de SDL_TTF y la librería de compresión ZLib. Todas estas librerías se incluyen junto a la distribución de código del juego para Windows. En Linux, el sistema donde se compile debe tener los paquetes de dichas librerías instaladas. Junto a la distribución de código, se incluye también un archivo makefile para facilitar la compilación. Para ejecutarlo, se debe lanzar el comando 'make target=N' en el directorio del videojuego, donde N puede ser 'windows' o 'linux', dependiendo cual sea el sistema operativo.

2.EL JUEGO

Una vez compilado, o desempaqueto una distribución binaria del juego ya es posible utilizarlo.Se puede ejecutar el juego en modo pantalla completa llamando al programa con el par√°metro -f, '$./coconut -f'. La primera pantalla que se visualizará es un pequeño menú con las opciones de 'nueva partida', 'información' y salir. La pantalla de juego es como la siguiente:

Durante el juego se maneja el 'coco' mediante las teclas del cursor. La misión del juego es hacer que el este personaje se coma todas las estrellas del escenario y antes de que se agote el tiempo de la partida. Además, aparecen unos fantasmas por el escenario, si alguno de ellos tocan a nuestro personaje, éste perderá una vida. En los escenarios también suelen aparecer caramelos, si el coco se come alguno de ellos tendrá la habilidad de comerse también a los fantasmas y ser invulnerable durante un tiempo. En la parte inferior de la ventana del juego se muestra una barra de información, la cual muestra las estrellas que faltan por comer, el tiempo restante del escenario, las vidas que le quedan a nuestro personaje y la puntuación total de la partida.
